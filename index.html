WRF-Chem-Polar

A derived version of the WRF-Chem regional chemistry-climate model, with enhancements for the polar regions, developed at IGE and LATMOS in France.

Our two main Git repositories:

  • The model code of WRF-Chem-Polar
  • Our infrastructure code, which we use for:
    • Compiling WPS and WRF-Chem
    • Preprocessing data
    • Running the model
    • Post-processing model outputs
